# Wood App

Корпоративное веб-приложение для каталога акустических панелей с клиентской витриной, личным кабинетом, административной панелью, CRM-синхронизацией и веб-аналитикой.

## Структура проекта

- `frontend/` — клиентская часть на `Next.js + TypeScript`
- `backend/` — серверная часть на `FastAPI + Python`
- `docs/` — схемы, технические материалы и служебная документация
- `supplementary-materials/` — архивные файлы, исходные ассеты и дополнительные материалы, не участвующие в рабочем запуске приложения

## Возможности

- каталог коллекций и карточки товаров
- корзина, избранное и личный кабинет
- оформление заказов и отправка заявок
- административная панель с CMS-функциями
- управление товарами из админки
- интеграция с Bitrix24
- подключение Яндекс.Метрики и фиксация пользовательских событий

## Быстрый запуск

### Frontend

```powershell
cd frontend
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`.

### Backend

```powershell
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
.venv\Scripts\uvicorn.exe app.main:app --reload
```

API будет доступно по адресу `http://localhost:8000`.

## Интеграции

- `Bitrix24` настраивается из административной панели через webhook, pipeline ID и responsible ID
- `Яндекс.Метрика` подключается через ID счетчика и флаг включения в настройках сайта

## Примечание

Рабочая часть проекта собрана внутри каталога `project/`, а дополнительные материалы вынесены в `supplementary-materials/`, чтобы структура оставалась чистой, понятной и удобной для сопровождения.
